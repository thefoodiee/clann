# Clann

A streamlined, community-centric platform designed to simplify group management and foster real-time connection. **Clann** empowers users to build organized digital spaces for their circles, whether for professional networking or casual social groups.

---

## 🚀 Features

- 🌐 **Community Creation**: Easily set up and customize your "Clann" with unique branding.
- 💬 **Class Feed**: Feed post functionality to keep classmates engaged and informed.
- 🔐 **Role Management**: Assign class representative roles to maintain order within the group.
- 📱 **Responsive UI**: A clean, modern interface designed for seamless navigation across all devices.

---

## 🛠️ Tech Stack

- **Frontend**: Flutter / Dart
- **Backend**: Supabase
- **Database**: Supabase Database
- **Authentication**: Supabase Auth
